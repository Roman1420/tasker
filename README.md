
# Tasker

It's simple project - board for managing your tasks in browser.
    

## Run Locally

Clone the project

```bash
  git clone https://github.com/Roman1420/tasker.git
```

Go to the project directory

```bash
  cd tasker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


Finished!
